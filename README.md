## React Finite State Machine Component

## Install
```
npm install --save react-fsm-component
```

## Build
```
git clone https://github.com/nhz-io/react-fsm-component.git
cd react-fsm-component
npm install
npm start
...
check http://localhost:9000
...
npm dist
```
